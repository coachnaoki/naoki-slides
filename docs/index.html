Naoki式 AI スライド自動生成テンプレート

台本テキストを渡すだけで Google Slides が完成する naoki-slides のセットアップガイド

⚠ 前提：naoki-blueprint 生徒限定

本テンプレは naoki-blueprint 生徒限定です。ライセンスIDは naoki-blueprint で発行されたものをそのまま使います（新規に ID は発行しません）。

naoki-blueprint を未導入の方は、まず naoki-blueprint のセットアップ を先にお願いします。

1 リポジトリをダウンロード

⚠ 事前準備: すでに Claude Code を開いている方は、ターミナル上で Ctrl+C（Mac は Control+C）を 2回押して終了してから進めてください。

Cursor のターミナルを開き、naoki-slides を置きたいフォルダに移動します（naoki-blueprint と同じ階層に置くとライセンスを自動で引き継げます）。

① 置きたいフォルダに移動（~/Desktop/Cursor は例。お好きな場所でOK）

cd ~/Desktop/Cursor

② リポジトリをダウンロード

git clone https://github.com/coachnaoki/naoki-slides.git

③ ダウンロードしたフォルダに入る

cd naoki-slides
以降のコマンドは Mac / Windows 共通です。Cursor のターミナルで、そのままコピペして実行してください。

2 初回セットアップ（Google Slide + GAS）

初回だけ、自分の Drive に Slide テンプレと GAS スクリプトをコピーします。Cursor のターミナルで以下を実行：

bash セットアップ.sh

対話形式で以下を案内されます：

Step 2-1. Slide テンプレを自分の Drive にコピー

  1. 表示されたリンクを開く → 「コピーを作成」 を押す（自分の Drive に Slide + GAS が一緒にコピーされる）
  2. コピー後に開いた Slide の URL をコピー
  3. ターミナルに URL を貼り付けて Enter
⚠ このSlideは「テンプレ専用」として使われます。実際のプレゼンを作るときは、このテンプレから毎回自動でコピーが作られるので、中身を書き換えても構いません（上書きされません）。

Step 2-2. GAS をデプロイして URL を取得

  1. Step 2-1 でコピーした Slide の上部メニュー 「拡張機能」→「Apps Script」
  2. 画面右上の 「デプロイ」→「新しいデプロイ」
  3. 歯車アイコン → 「ウェブアプリ」 を選択
  4. 次のユーザーとして実行: 「自分」
  5. アクセスできるユーザー: 「全員」（「自分のみ」だと動きません）
  6. 「デプロイ」 を押す
  7. 初回は承認ダイアログ：「詳細」→「{プロジェクト名}（安全ではないページ）に移動」→ 許可
  8. 表示された 「ウェブアプリ URL」 をコピー
  9. ターミナルに URL を貼り付けて Enter
なぜ「全員」？ Claude がリクエストを送るため。URL は推測困難なので、セキュリティ上は問題ありません。
完了すると .env が自動生成されます。詰まったら次のステップで Claude Code を起動して聞いてください。

3 プレゼンを作成して台本を書く

ターミナルで以下を実行：

bash 新規スライド.sh

プレゼン名を聞かれるので入力（例: tennis-camp-2026）。
→ projects/tennis-camp-2026/script/台本.md が作られます。

Cursor のエディタで 台本.md を開き、動画や記事の台本をそのまま貼り付けるだけで OK。

  • 構成や見出しは考えなくて大丈夫（Claudeが自動で整えます）
  • 話し言葉・箇条書き・混在でもOK
  • ファイル形式は自由（.md / .txt / 他）

4 Claude Code を立ち上げる

作成したプレゼンフォルダに移動してから Claude Code を起動します。この順番が重要（ルートで先に起動すると cwd がズレてスキルが動きません）。

cd projects/tennis-camp-2026
claude --dangerously-skip-permissions
tennis-camp-2026 の部分は Step 3 で付けたプレゼン名に置き換えてください。
初回起動時にログインを求められた場合は、画面の指示に従ってブラウザで認証してください。

5 スライドを生成する

Step 4 で立ち上げた Claude Code にそのまま以下を入力：

/slides-create

スキルが自動で以下をすべて実行してくれます：

  1. 依存パッケージのインストール（npm install）
  2. ライセンス認証（naoki-blueprint から自動コピー）
  3. 台本を解析してスライド構成を設計
  4. 構成案を提示 → あなたが OK と言うまで JSON 確定しない
  5. Puppeteer で画像化（1280×720 PNG × N枚）
  6. プレビューをブラウザで自動表示 → 目視確認
  7. OK なら GAS 経由で Google Slide に反映
  8. 完成 URL が返ってくる → ブラウザで開いて確認
約1分で完成。編集したい場合は script/ 内の台本を修正して /slides-create を再実行してください。

2回目以降（新しいプレゼンを作るとき）

Step 1（ダウンロード）と Step 2（Google Slide + GAS）は初回だけでOK。
2回目以降は /slides-create がプレゼン作成からスライド生成まで全部やってくれます。

① naoki-slides ディレクトリに移動して Claude Code を起動（~/Desktop/Cursor は clone 先に合わせて読み替え）

cd ~/Desktop/Cursor/naoki-slides
claude --dangerously-skip-permissions

② スキルを起動

/slides-create

あとは対話で進みます：

  1. 「新規」か「既存」かを聞かれる → 新規を選ぶ
  2. プレゼン名を入力（例: tennis-camp-2026）
  3. Claude が自動で以下を実行： ①プロジェクトフォルダ作成 / ②新しいGoogle Slideを作成（テンプレから自動コピー）
  4. projects/{プレゼン名}/script/台本.md が作られるので、Cursor で開いて台本を貼り付ける
  5. チャットに「続けて」と入力 → そのままスライド設計・生成・反映まで自動進行
⚠ 過去のプレゼンは上書きされません。 新規作成ごとに独立した Google Slide が自動で生成されます（Step 1 で作ったSlideはテンプレ専用）。
ライセンス認証・.env 設定は初回に済んでいるので、そのまま使えます。

アップデート（バージョンアップ）

テンプレートが更新されたら、以下のコマンドで最新版に置き換えできます。
あなたが作った projects/ と .license / .env は上書きされず安全です。

① naoki-slides ディレクトリに移動（~/Desktop/Cursor は clone 先に合わせて読み替え）

cd ~/Desktop/Cursor/naoki-slides

② アップデートスクリプトを実行

bash アップデート.sh

裏側で走っている処理（参考）

フェーズ1: 設計
台本解析Claude が章立て・数字・キーワードを抽出
テンプレート選択17種から最適なテンプレを自動選択
構成案提示あなたに OK を貰う（必須）
↓
フェーズ2: 生成
JSON 生成slide-data.json を書き出し
Puppeteer 画像化HTMLテンプレを 1280×720 PNG にキャプチャ
プレビュー表示ブラウザで全枚数を縦並びに表示
↓
フェーズ3: 反映
GAS へ POSTBase64化した画像を Webhook で送信
Slide 書き換えGAS が既存スライドを全削除 → 画像を貼り付け
完成Google Slide の URL が返る

17種類のテンプレート（自動選択）

Claude が台本の内容から適切なテンプレートを自動選択します。

rankingランキング

highlight-box強調ボックス
quote引用・名言
closing締め・CTA

フォルダ構成

naoki-slides/                     ← ダウンロードしたフォルダ
├── README.md                     ← ガイド
├── セットアップ.sh                 ← 初回セットアップ（.env 自動生成）
├── 新規スライド.sh                 ← 新規プレゼン作成
├── アップデート.sh                 ← 最新版に更新
├── .claude/skills/slides-create/  ← メインスキル
├── .template-project/             ← 新規プレゼンの雛形
│   └── script/台本.md
├── templates/slides.html          ← 17種テンプレ
├── tools/                         ← Node/JSユーティリティ
│   ├── validateLicense.mjs
│   ├── find-license.mjs
│   ├── screenshot-slides.mjs
│   ├── generate-preview.mjs
│   └── post-to-gas.mjs
├── gas/                           ← GAS本体（参考）
│
└── projects/                      ← あなたのプレゼン（gitignore）
    └── {プレゼン名}/
        ├── script/台本.md          ← 台本（形式自由）
        ├── slide-data.json         ← Claudeが生成
        └── output/                 ← スライド画像
